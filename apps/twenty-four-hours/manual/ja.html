Twenty Four Hours

ユーザーマニュアル

アプリ情報

アプリ名Twenty Four Hours
バージョン1.0.0
開発Willee Project
対応プラットフォームAndroid, Windows

24時間を一目で確認できるアナログ時計とスケジュール管理アプリです。

メイン画面

アナログ時計

  • 24時間形式のアナログ時計が画面上部に表示されます
  • スケジュールは時計の背景に色付きのアーク（弧）で表示されます
  • スケジュールのアークをタップすると詳細情報が表示されます
  • 時計をダブルタップするとフルスクリーンモードに切り替わります

フルスクリーンモード

  • ダブルタップで開始/終了
  • スケジュールのアークをタップすると右下に詳細カードが表示されます
  • 左右スワイプ：時計スタイル変更（ミニマル ↔ クラシック）
  • 上下スワイプ：秒針スタイル変更（スムーズ ↔ チクタク）

デジタル時計

  • 上部のアプリバーに現在時刻がHH:MM:SS形式で表示されます

スケジュールリストパネル

  • 画面下部で現在と今日のスケジュールを確認できます
  • 現在：今進行中のスケジュール
  • 今日：今日のすべてのスケジュール
  • 左右スワイプまたは上部ボタンで切り替えられます
  • ➕ボタンで新しいスケジュールを追加できます

スケジュール管理

新規スケジュール作成

  1. メイン画面の➕ボタンまたはスケジュール管理画面の➕ボタンをタップ
  2. タイトルを入力（必須、前後の空白は自動的に削除されます）
  3. 詳細を入力（任意）
  4. 開始/終了の日付と時刻を設定
  5. 必要に応じて終日、繰り返し、重要度、色を設定
  6. 保存ボタンをタップ

終日イベント

  • 終日トグルをオンにすると、時刻指定なしの終日スケジュールになります
  • 複数日にわたる終日スケジュールも可能です

繰り返しスケジュール

  • 繰り返しタイプ：毎日、毎週、毎月、毎年
  • 繰り返し終了：続ける（無期限）、日付まで、回数
  • 毎週の繰り返し：複数の曜日を選択できます（デフォルト：開始曜日）
    • すべての曜日を選択解除できます。曜日が選択されていない状態で保存すると警告が表示されます
  • 注意：スケジュールの期間が繰り返し周期より長い場合、その繰り返しタイプは選択できません
    • 例：26時間のスケジュール → 毎日繰り返し不可
    • 例：8日間のスケジュール → 毎週繰り返し不可
    • 期間変更により繰り返し設定が解除された場合、警告メッセージが表示されます
    • 警告状態で保存すると確認ダイアログが表示されます
  • 編集時の達成記録：繰り返しスケジュール編集時、過去の達成記録は保持され、無効な未来の記録のみ整理されます

重要度設定

  • ★ 自分らしく：続けるだけで十分です
  • ★★ いい調子：やり遂げれば今日が輝きます
  • ★★★ あなたならできる：きっと誇りに思えるはず

完了チェック

  • スケジュールリストのチェックボックスをタップして完了マークを付けられます
  • 完了したスケジュールは取り消し線で表示されます
  • 繰り返しスケジュールは各発生日ごとに個別にチェックできます

達成記録管理

  • ⋮メニューから「達成記録」を選択
  • 過去と未来の日付の完了状態を管理できます

スケジュールをコピー

  • ⋮メニューから「スケジュールをコピー」を選択
  • すべての設定がコピーされた状態でスケジュールフォームが開きます
  • 必要に応じて修正し、保存すると新しいスケジュールとして登録されます

統計

統計画面でスケジュールの達成率を確認できます。

期間選択

  • 今日、今週、今月、今年、期間指定

計算範囲

  • 今日まで：選択した期間のうち今日までを計算（デフォルト）
  • 全期間：未来のスケジュールも含めて計算

統計情報

  • 全体の進捗とフィードバックメッセージ
  • 重要度別の達成率
  • 詳細スケジュールリスト（展開可能）

設定

テーマ

  • システム設定、ライトモード、ダークモード

時計スタイル

  • ミニマル：主要な時刻のみ表示（0, 6, 12, 18）
  • クラシック：すべての時刻を表示（0-23）、外周に5分単位の点を表示

秒針スタイル

  • スムーズ：滑らかに動く秒針
  • チクタク：1秒ごとに動く秒針

言語

  • 自動（システム）、한국어、English、日本語、中文

ディスプレイ（モバイル）

  • 画面を点灯したままにする：アプリ実行中、画面が自動でオフになりません

Windows

  • 常に手前に表示：他のウィンドウより前に表示されます

スケジュール管理

  • 過去のスケジュールのグループ化：なし、月別、年+月別

クラウド同期

Googleアカウントでログインしてデータをクラウドにバックアップできます。

基本機能

  • 複数デバイス間でデータを同期
  • アプリ再インストール時にデータ復元可能

同期方式

  • 自動同期: データ変更後30秒後に自動的に同期されます
  • 手動同期: 設定画面で「手動同期」ボタンをタップして即座に同期

競合の処理

  • 複数のデバイスで同時に変更した場合、競合が発生する可能性があります
  • 手動同期時に競合が検出されると、選択画面が表示されます：
    • このデバイスを維持: このデバイスのデータでクラウドを上書きします
    • クラウドを維持: クラウドのデータでこのデバイスを上書きします
    • マージ: 両方のデータを統合し、各項目の最新バージョンを維持します（推奨）
  • 自動同期では競合時に自動的にマージされます

クラウドをリセット

  • 同期に問題が発生した場合、「クラウドをリセット」機能を使用できます
  • クラウドのすべてのデータを削除し、現在のデバイスのデータを新たにアップロードします

ヒント

  • 時計をダブルタップするとフルスクリーンモードに切り替わります
  • フルスクリーンでスワイプすると時計/秒針スタイルを変更できます
  • モバイルで画面を横向きにすると時計とスケジュールリストが左右に分かれます
  • スケジュールが重なると最大3レイヤーで表示され、4つ以上は「+N」バッジで表示されます

アプリ開発を応援する

アプリ開発を応援していただくと広告が永久に削除されます。（Androidのみ）
