사용 설명서

앱 정보

앱 이름Twenty Four Hours
버전1.0.0
개발Willee Project
지원 플랫폼Android, Windows

24시간을 한눈에 볼 수 있는 아날로그 시계와 일정 관리 앱입니다.

메인 화면

아날로그 시계

  • 24시간 형식의 아날로그 시계가 화면 상단에 표시됩니다
  • 일정이 있으면 시계 배경에 색상 호(arc)로 표시됩니다
  • 일정 호를 탭하면 해당 일정의 상세 정보를 볼 수 있습니다
  • 시계를 더블 탭하면 전체 화면 모드로 전환됩니다

전체 화면 모드

  • 시계를 더블 탭하여 진입/종료
  • 일정 호를 탭하면 우측 하단에 상세 정보 카드가 표시됩니다
  • 좌우 스와이프: 시계 스타일 변경 (미니멀 ↔ 클래식)
  • 상하 스와이프: 초침 스타일 변경 (스무스 ↔ 틱톡)

디지털 시계

  • 상단 앱바에 현재 시간이 HH:MM:SS 형식으로 표시됩니다

일정 목록 패널

  • 화면 하단에 현재 일정과 오늘 일정을 확인할 수 있습니다
  • 현재: 지금 진행 중인 일정
  • 오늘: 오늘 하루의 모든 일정
  • 좌우 스와이프 또는 상단 버튼으로 전환할 수 있습니다
  • ➕ 버튼을 눌러 새 일정을 추가할 수 있습니다

일정 관리

새 일정 만들기

  1. 메인 화면의 ➕ 버튼 또는 일정 관리 화면에서 ➕ 버튼을 탭합니다
  2. 제목을 입력합니다 (필수, 좌우 공백은 자동으로 제거됩니다)
  3. 상세 내용을 입력합니다 (선택)
  4. 시작/종료 날짜와 시간을 설정합니다
  5. 필요시 종일, 반복, 중요도, 색상을 설정합니다
  6. 저장 버튼을 탭합니다

종일 일정

  • 종일 토글을 켜면 시간 없이 하루 종일 일정으로 설정됩니다
  • 여러 날에 걸친 종일 일정도 가능합니다

반복 일정

  • 반복 유형: 매일, 매주, 매월, 매년
  • 반복 종료: 계속(무기한), 특정 날짜까지, 특정 횟수
  • 매주 반복: 여러 요일을 선택할 수 있습니다 (기본값: 시작 요일)
    • 모든 요일을 선택 해제할 수 있으며, 요일 미선택 시 저장할 때 알림이 표시됩니다
  • 참고: 일정 기간이 반복 주기보다 길면 해당 반복 유형은 선택할 수 없습니다
    • 예: 26시간 일정 → 매일 반복 불가
    • 예: 8일 일정 → 매주 반복 불가
    • 일정 기간 변경으로 반복 설정이 해제되면 경고 메시지가 표시됩니다
    • 경고 상태에서 저장 시 확인 다이얼로그가 표시됩니다
  • 수정 시 완료 기록: 반복 일정 수정 시 과거 완료 기록은 유지되고, 미래의 무효한 기록만 정리됩니다

중요도 설정

  • ★ 그냥 너답게: 꾸준히 하는 것만으로 충분해
  • ★★ 잘 하고 있어: 해내면 오늘 하루가 빛날 거야
  • ★★★ 넌 할 수 있어: 해내면 정말 자랑스러울 거야

일정 완료 체크

  • 일정 목록에서 체크박스를 탭하여 완료 표시를 할 수 있습니다
  • 완료된 일정은 취소선으로 표시됩니다
  • 반복 일정은 각 발생일마다 개별적으로 완료 체크가 가능합니다

달성 기록 관리

  • 일정의 ⋮ 메뉴에서 "달성 기록"을 선택합니다
  • 과거 및 미래 날짜의 완료 상태를 관리할 수 있습니다

일정 복사

  • 일정의 ⋮ 메뉴에서 "일정 복사"를 선택합니다
  • 일정 편집 화면이 열리며, 모든 설정이 복사된 상태로 표시됩니다
  • 필요한 부분을 수정 후 저장하면 새 일정으로 등록됩니다

통계

통계 화면에서 일정 달성률을 확인할 수 있습니다.

기간 선택

  • 오늘, 이번 주, 이번 달, 올해, 사용자 지정 기간

계산 범위

  • 오늘까지: 선택한 기간 중 오늘까지만 계산 (기본값)
  • 전체 기간: 미래 일정도 포함하여 계산

통계 정보

  • 전체 진행률과 피드백 메시지
  • 중요도별 달성률
  • 상세 일정 목록 (펼쳐서 확인 가능)

설정

테마

  • 시스템 설정 따름, 라이트 모드, 다크 모드

시계 스타일

  • 미니멀: 주요 시간만 표시 (0, 6, 12, 18)
  • 클래식: 모든 시간 표시 (0-23), 외곽에 5분 단위 점 표시

초침 스타일

  • 스무스: 부드럽게 움직이는 초침
  • 틱-톡: 1초마다 움직이는 초침

언어

  • 자동(시스템), 한국어, English, 日本語, 中文

화면 (모바일)

  • 화면 켜짐 유지: 앱 실행 중 화면이 자동으로 꺼지지 않습니다

Windows

  • 항상 위에 표시: 다른 창 위에 항상 표시됩니다

일정 관리

  • 지난 일정 그룹화: 없음, 월별, 연도+월별

개발 응원하기

앱 개발을 응원해 주시면 광고가 영구적으로 제거됩니다. (Android만 지원)

클라우드 동기화

Google 계정으로 로그인하여 데이터를 클라우드에 백업할 수 있습니다. (Android만 지원)

  • 여러 기기 간 데이터 동기화
  • 수동 동기화 방식
  • 앱 재설치 시 데이터 복원 가능

팁

  • 시계를 더블 탭하면 전체 화면 모드로 전환됩니다
  • 전체 화면에서 스와이프로 시계/초침 스타일을 변경할 수 있습니다
  • 모바일에서 화면을 가로로 회전하면 시계와 일정 목록이 좌우로 나뉩니다
  • 일정이 겹치면 최대 3개 레이어로 표시되고, 4개 이상은 "+N" 배지로 표시됩니다
