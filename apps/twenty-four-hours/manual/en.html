Twenty Four Hours

User Manual

App Information

App NameTwenty Four Hours
Version1.0.0
DevelopmentWillee Project
PlatformsAndroid, Windows

An analog clock app that shows 24 hours at a glance with schedule management.

Main Screen

Analog Clock

  • A 24-hour analog clock is displayed at the top of the screen
  • Schedules are shown as colored arcs on the clock background
  • Tap a schedule arc to view its details
  • Double-tap the clock to enter full-screen mode

Full-Screen Mode

  • Double-tap to enter/exit
  • Tap a schedule arc to show details card at bottom-right
  • Horizontal swipe: Change clock style (Minimal ↔ Classic)
  • Vertical swipe: Change second hand style (Smooth ↔ Tick-Tock)

Digital Clock

  • Current time is displayed in HH:MM:SS format in the app bar

Schedule List Panel

  • View current and today's schedules at the bottom of the screen
  • Current: Schedules in progress right now
  • Today: All schedules for today
  • Swipe left/right or use the buttons to switch
  • Tap ➕ to add a new schedule

Schedule Management

Creating a New Schedule

  1. Tap the ➕ button on the main screen or schedule list screen
  2. Enter a title (required, leading/trailing spaces are automatically removed)
  3. Enter description (optional)
  4. Set start/end date and time
  5. Set all-day, repeat, priority, and color as needed
  6. Tap Save

All-Day Events

  • Turn on the all-day toggle to create a schedule without specific times
  • Multi-day all-day events are supported

Repeating Schedules

  • Repeat type: Daily, Weekly, Monthly, Yearly
  • Repeat end: Never, Until date, After count
  • Weekly repeat: You can select multiple days of the week (default: start day)
    • All days can be deselected; saving without any day selected will show a warning
  • Note: If the schedule duration exceeds the repeat cycle, that repeat type is disabled
    • e.g., 26-hour schedule → Daily repeat unavailable
    • e.g., 8-day schedule → Weekly repeat unavailable
    • A warning message appears if repeat settings are reset due to duration change
    • A confirmation dialog appears when saving with the warning active
  • Completion records on edit: Past completion records are preserved, only invalid future records are cleaned up

Priority Levels

  • ★ Just be yourself: Being consistent is enough
  • ★★ You're doing great: Complete it and your day will shine
  • ★★★ You can do it: You'll be so proud when you finish

Completion Tracking

  • Tap the checkbox in the schedule list to mark as complete
  • Completed schedules are shown with a strikethrough
  • Repeating schedules can be checked individually for each occurrence

Completion Records

  • Select "Completion Records" from the ⋮ menu
  • Manage completion status for past and future dates

Copy Schedule

  • Select "Copy Schedule" from the ⋮ menu
  • The schedule form opens with all settings copied
  • Modify as needed and save to create a new schedule

Statistics

View your schedule completion rates in the Statistics screen.

Period Selection

  • Today, This Week, This Month, This Year, Custom Period

Calculation Scope

  • Until Today: Calculate only up to today (default)
  • Entire Period: Include future schedules

Statistics Info

  • Overall progress with feedback message
  • Completion rate by priority
  • Detailed schedule list (expandable)

Settings

Theme

  • System Default, Light Mode, Dark Mode

Clock Style

  • Minimal: Show main hours only (0, 6, 12, 18)
  • Classic: Show all hours (0-23), with 5-minute dots on the outer edge

Second Hand Style

  • Smooth: Smoothly moving second hand
  • Tick-Tock: Second hand moves every second

Language

  • Auto (System), 한국어, English, 日本語, 中文

Display (Mobile)

  • Keep Screen On: Screen won't turn off while the app is running

Windows

  • Always on Top: Window stays above other windows

Schedule Management

  • Past Schedule Grouping: None, Monthly, Year + Monthly

Cloud Sync

Sign in with Google to backup your data to the cloud.

Basic Features

  • Sync data across multiple devices
  • Restore data after reinstalling the app

Sync Methods

  • Auto Sync: Automatically syncs 10 seconds after data changes
  • Manual Sync: Tap "Manual sync" button in settings to sync immediately

Conflict Resolution

  • Conflicts may occur when changes are made on multiple devices simultaneously
  • When a conflict is detected during manual sync, a selection screen appears:
    • Keep This Device: Overwrite cloud with this device's data
    • Keep Cloud: Overwrite this device with cloud data
    • Merge: Combine both, keeping the newest version of each item (Recommended)
  • Auto sync automatically merges when conflicts occur

Reset Cloud Data

  • Use "Reset cloud data" if sync issues occur
  • Deletes all cloud data and uploads current device data as fresh backup

Tips

  • Double-tap the clock to enter full-screen mode
  • Swipe in full-screen to change clock/second hand styles
  • Rotate your mobile device to landscape for a split-screen layout
  • Overlapping schedules show up to 3 layers, with "+N" badge for 4 or more
