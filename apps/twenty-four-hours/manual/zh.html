应用信息

应用名称Twenty Four Hours
版本1.0.0
开发Willee Project
支持平台Android, Windows

一款可以一目了然查看24小时的模拟时钟和日程管理应用。

主界面

模拟时钟

  • 24小时制模拟时钟显示在屏幕顶部
  • 日程以彩色弧形显示在时钟背景上
  • 点击日程弧形可查看详细信息
  • 双击时钟进入全屏模式

全屏模式

  • 双击进入/退出
  • 点击日程弧形，右下角显示详情卡片
  • 左右滑动：切换时钟样式（简约 ↔ 经典）
  • 上下滑动：切换秒针样式（流畅 ↔ 滴答）

数字时钟

  • 顶部应用栏以 HH:MM:SS 格式显示当前时间

日程列表面板

  • 在屏幕底部查看当前和今日日程
  • 当前：正在进行的日程
  • 今天：今天的所有日程
  • 左右滑动或点击按钮切换
  • 点击 ➕ 添加新日程

日程管理

创建新日程

  1. 点击主界面或日程管理界面的 ➕ 按钮
  2. 输入标题（必填，首尾空格会自动删除）
  3. 输入详细内容（可选）
  4. 设置开始/结束日期和时间
  5. 根据需要设置全天、重复、重要程度和颜色
  6. 点击保存

全天日程

  • 开启全天开关，创建没有具体时间的全天日程
  • 支持跨多天的全天日程

重复日程

  • 重复类型：每天、每周、每月、每年
  • 重复结束：永不、直到日期、次数
  • 每周重复：可以选择多个星期几（默认：开始日的星期几）
    • 可以取消选择所有星期，如果没有选择任何星期保存时会显示提示
  • 注意：如果日程时长超过重复周期，该重复类型将被禁用
    • 例：26小时日程 → 无法选择每天重复
    • 例：8天日程 → 无法选择每周重复
    • 如果因时长变更导致重复设置被取消，将显示警告消息
    • 在警告状态下保存时将显示确认对话框
  • 编辑时的完成记录：编辑重复日程时，过去的完成记录会保留，仅清理无效的未来记录

重要程度设置

  • ★ 做自己就好：坚持就足够了
  • ★★ 你做得很棒：完成它，今天会更美好
  • ★★★ 你可以的：完成后你会为自己骄傲

完成打卡

  • 点击日程列表中的复选框标记完成
  • 已完成的日程显示删除线
  • 重复日程可以为每个发生日单独打卡

完成记录管理

  • 从 ⋮ 菜单选择"完成记录"
  • 管理过去和未来日期的完成状态

复制日程

  • 从 ⋮ 菜单选择"复制日程"
  • 日程表单将打开，所有设置已复制
  • 根据需要修改后保存，即可创建新日程

统计

在统计界面查看日程完成率。

期间选择

  • 今天、本周、本月、今年、自定义

计算范围

  • 截至今天：只计算到今天为止（默认）
  • 整个期间：包括未来日程

统计信息

  • 整体进度和反馈消息
  • 按重要程度的完成率
  • 详细日程列表（可展开）

设置

主题

  • 跟随系统、浅色模式、深色模式

时钟样式

  • 简约：仅显示主要时刻（0, 6, 12, 18）
  • 经典：显示所有时刻（0-23），外圈显示5分钟刻度点

秒针样式

  • 流畅：秒针平滑移动
  • 滴答：秒针每秒跳动

语言

  • 自动（系统）、한국어、English、日本語、中文

显示（移动端）

  • 保持屏幕常亮：应用运行时屏幕不会自动关闭

Windows

  • 始终置顶：窗口始终显示在最前面

日程管理

  • 过去日程分组：无、按月、按年+月

支持应用开发

支持应用开发后将永久移除广告。（仅限 Android）

云同步

使用 Google 账号登录，将数据备份到云端。（仅限 Android）

  • 多设备间数据同步
  • 手动同步方式
  • 重新安装应用后可恢复数据

小贴士

  • 双击时钟进入全屏模式
  • 全屏模式下滑动可更改时钟/秒针样式
  • 移动设备横屏时，时钟和日程列表左右分开显示
  • 重叠日程最多显示3层，4个以上显示"+N"徽章
