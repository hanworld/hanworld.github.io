최종 업데이트: 2025년 1월 26일

Willee Project(이하 "개발자")는 Twenty Four Hours(이하 "앱") 이용자의 개인정보를 중요하게 생각하며, 개인정보보호법 등 관련 법령을 준수하고 있습니다.

1. 수집하는 개인정보

1.1 필수 수집 정보

앱의 기본 기능 사용을 위해 수집되는 정보는 없습니다. 일정 데이터는 사용자의 기기에만 저장됩니다.

1.2 선택적 수집 정보

항목수집 목적수집 시점
Google 계정 정보 (이메일)클라우드 동기화 서비스 제공사용자가 동기화 기능 사용 시
광고 식별자맞춤형 광고 제공앱 실행 시 (광고 표시 시)
구매 정보인앱 결제 처리 및 광고 제거인앱 결제 진행 시

2. 개인정보의 저장 및 보관

2.1 로컬 저장

일정 데이터, 완료 기록, 앱 설정은 사용자의 기기에 로컬로 저장되며, 개발자의 서버로 전송되지 않습니다.

2.2 클라우드 저장 (선택)

사용자가 클라우드 동기화 기능을 활성화한 경우, 데이터는 사용자 본인의 Google Drive에 저장됩니다. 개발자는 이 데이터에 접근할 수 없습니다.

3. 제3자 서비스

앱은 다음과 같은 제3자 서비스를 사용합니다:

  • Google AdMob: 광고 제공 (Google 개인정보처리방침)
  • Google Play 결제: 인앱 결제 처리
  • Google Drive API: 클라우드 동기화 (선택)
  • Google Sign-In: 계정 인증 (선택)

4. 개인정보의 보유 및 파기

  • 로컬 데이터: 앱 삭제 시 자동으로 삭제됩니다.
  • 클라우드 데이터: 사용자가 Google Drive에서 직접 삭제할 수 있습니다.
  • 광고 식별자: 기기 설정에서 재설정하거나 비활성화할 수 있습니다.

5. 이용자의 권리

이용자는 다음과 같은 권리를 가집니다:

  • 클라우드 동기화 기능 사용 여부 선택
  • 인앱 결제를 통한 광고 제거
  • 앱 삭제를 통한 모든 로컬 데이터 삭제
  • Google 계정 연결 해제

6. 아동의 개인정보

이 앱은 만 13세 미만의 아동을 대상으로 하지 않으며, 의도적으로 아동의 개인정보를 수집하지 않습니다.

7. 계정 및 데이터 삭제

사용자는 언제든지 자신의 데이터를 삭제할 수 있습니다.

7.1 로컬 데이터 삭제

기기에 저장된 모든 데이터(일정, 완료 기록, 설정)는 앱을 삭제하면 자동으로 삭제됩니다.

  • Android: 설정 → 앱 → Twenty Four Hours → 삭제
  • Windows: 앱 폴더를 삭제하거나 제어판에서 제거

7.2 클라우드 데이터 삭제

Google Drive에 저장된 백업 데이터를 삭제하려면:

  • Google Drive에 접속
  • "내 드라이브" → 앱 데이터 폴더에서 "twenty_four_hours_backup.json.gz" 파일 삭제
  • 또는 Google 계정 권한 관리에서 앱 연결 해제

7.3 Google 계정 연결 해제

앱에서 Google 계정 연결을 해제하려면:

  • 앱 내 설정 → 클라우드 동기화 → 로그아웃
  • 또는 Google 계정 권한 관리에서 "Twenty Four Hours" 앱 액세스 권한 삭제

7.4 데이터 삭제 요청

위 방법으로 삭제가 어려운 경우, 아래 문의처로 연락해 주시면 삭제를 도와드리겠습니다.

8. 개인정보처리방침의 변경

본 개인정보처리방침은 법령 또는 서비스 변경에 따라 수정될 수 있습니다. 중요한 변경사항이 있을 경우 앱 내 공지를 통해 안내드립니다.
